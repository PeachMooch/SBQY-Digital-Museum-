# 农具图片目录

这个目录包含农具演进展示所需的图片资源。

## 图片列表

- **stone.png**: 石碾图片（明朝）
- **wood.png**: 木犁图片（清朝）
- **iron.png**: 铁犁图片（民国）
- **drone.png**: 现代无人机图片（当代）

## 图片要求

1. **格式**: PNG格式，透明背景
2. **尺寸**: 建议400x400px
3. **风格**: 简洁清晰，突出农具特征
4. **背景**: 透明背景，便于页面展示

## 设计建议

- 石碾：展现古朴的石器工艺
- 木犁：体现传统木工技艺
- 铁犁：展示金属工艺进步
- 无人机：现代科技感
